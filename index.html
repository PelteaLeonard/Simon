Instructions

Aim of the game

This game it's supposed to simulate the fizical Simon game.

How to play the game

You have a power switch where you turn the game on. When you press the power button you will see in count square a black line appears, then you know that the game is on.

You have a strict mode means that you have one wrong answer you'll lose all game.With strict mode activated it,s moore chalenging for you to play.

With strict mode off with one wrong answer you can start where you left off.

This is a memory matching game so you have to remember the colors of the game in exact order the game have swowed to you.

You have 20 rounds till you can finish the game. Good luck everyone!
